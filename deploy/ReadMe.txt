### Steps to execute the Product #####

Requirements
Java 7 or newer
Windows 7 or later

1) Unzip KafVam.zip in deploy folder to your localfile system
2) Unzip plugin1.zip and copy the plugins in plugin1 folder to plugins folder in KafVam folder
3) Unzip plugin2.zip and copy the plugins in plugin2 folder to plugins folder in KafVam folder
4) You can configure the default log4j.properties and KafVam.properties.
4) You are all set now. Run KafVam.exe and have a great day!!
