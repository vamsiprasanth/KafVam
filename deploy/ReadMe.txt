### Steps to execute the Product #####

Requirements
Java 7 or newer
Windows 7 or later

1) Unzip KafVam.zip in deploy folder to your localfile system
2) Unzip plugins1.zip and copy the plugins in plugins1 folder to plugins folder in KafVam folder
3) Unzip plugins2.zip and copy the plugins in plugins2 folder to plugins folder in KafVam folder
4) You can configure the default log4j.properties and KafVam.properties.The default logfile(KafVam.log) will be generated in KafVam folder.
5) You are all set now. Run KafVam.exe and have a great day!!
